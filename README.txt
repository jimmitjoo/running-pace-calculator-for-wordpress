A simple running pace calculator for WordPress.

== Description ==

This is a plugin that lets you add a running pace calculator on your WordPress website.

== Installation ==

1. Upload `running-pace-calculator` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place the shortcode `[running-pace-calculator]` wherever you want the calculator to appear.

== Frequently Asked Questions ==

= Does it calculate in miles or kilometers? =

Both, you can choose whichever you prefer in the admin area.
